# My Multipage Website

## Project Overview
This is a **responsive, interactive multipage website** built using **HTML5, CSS3, and JavaScript**.  
The project demonstrates my ability to plan, organize, code, and deploy a production-ready website.

**Website Purpose:**  
To showcase a simple portfolio/landing page with multiple sections and interactive elements.

---

## Pages Included
1. **Home** (`index.html`) – Welcome section with hero image and introduction.  
2. **About** (`about.html`) – Information about the website owner/project.  
3. **Contact** (`contact.html`) – Contact form with JavaScript validation.  

---

## Features
- **Responsive Design:** Works on desktop, tablet, and mobile devices.  
- **Navigation:** Consistent header and footer across all pages.  
- **Interactive Elements:**  
  - Mobile menu toggle  
  - Contact form validation using JavaScript  
- **Clean Code:** Semantic HTML5, modular CSS, and well-organized folder structure.
